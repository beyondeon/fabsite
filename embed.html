Embedded Programming

  • We have come to the last topic of the module. So, what is Embedded Programming? Simply put, embeddded programming is a form of coding / programming that is intended for, well, embedded systems. A common example of an embedded system would be the Arduino, which is coincidentally the microcontroller we will be using for the projects.
  • With embedded programming, we can code in commands and execute them on a controller, allowing it to carry out all kinds of functions depending on what modules are affixed.
  • The main form of code we use in Arduino is going to be C++, one of the many coding languages out there.
  • What we are going to use to simulate is TinkerCAD, which allows us to arrange components as well as execute code on the virtual Arduino.

Assignment Task 1

- The assignment is split into tasks, each one slowly increasing in complexity. You can view the code by clicking on the Code button, and try out the simulation by clicking "Start Simulation"

-These are the conditions for Task 1:

  1. Connect an LED with a current limiting resistor of 220 ohms to Port 6 of the Arduino.
  2. Write a program to blink the LED in a pattern.

- This program makes the LED blink once, twice, then thrice before looping.

Assignment Task 2

-These are the conditions for Task 2:

  1. Using the same techniques, connect 3 LEDs of different colours to the board using pins 6, 7, 8.
  2. Write a program to light up the LEDS in sequence (Red, Green, Blue, then loop)

- This program here lights up the LEDs in order, starting from Red all the way to Blue before turning them off in the same sequence and looping.

Assignment Task 3

-These are the conditions for Task 3:

  1. Add a button to the circuit in Task 1 (You should use a pull-up resistor of 10k ohms if operating in INPUT mode)
  2. Write a program that when the button is pressed, the LED turns on and when it is pressed again, it turns off.

- This code does just what is meant to do, turn an LED on and off when a button is pressed.

- The code implements a toggle latch into the button.

Assignment Task 4

-These are the conditions for Task 3:

  • Use the button in Task 3 to control the type of light up and flashing of the LEDs, for example:
  1. Start with all LEDs turned off
  2. Pressing the button again only turns on the Red LED
  3. Pressing the button again only turns on the Green LED
  4. Pressing the button again only turns on the Blue LED
  5. Pressing the button again turns on all the LEDs
  6. Pressing the button again turns off all the LEDs, returning to the first state.

- This code has a slight bug, where by you need to click the button twice to change the light states, and you also need to spam click the button to loop the sequence.

3D CAD

  • So what is 3D CAD? First off, CAD means "Computer Aided Drawing", and it is used to create and modify virtual models in both 3D and 2D. In this case, it would be well, 3D.
  • So just what is it used for? For one, 3D CAD is very useful and is basically one of the fundamentals to digital fabrication (hey that's the name of the module). 3D CAD is how we produce 3D models to print via 3D printers to realise our designs.
  • Various software for 3D CAD exist, with some examples being Autodesk Inventor and Autodesk Fusion 360. For our purposes, I will be using Fusion 360 for the majority of the module.
  • I have prior experience with CAD software as I am from an engineering course, namely Autodesk Inventor for 3D. However, Inventor and Fusion 360 have their own differences, so this is also a learning experience for me.

Exercise 1

- For this first 3D CAD exercise, we are actually not going to make a 3D model right away. We are going to make a sketch, which is the backbone of all 3D models in CAD software. Sketches allow designers to easily define dimensions and edit the shapes of the model. Only after making a sketch then are we able to extrude them in order to create a 3D model.

- We are going to sketch this in Fusion 360.

- First, we will click on the "Create Sketch" button, then select a plane to start the sketch on.

- Then, we can start sketching. I started with a rectangle. By default, the rectangle is a 2 point one, but by using the "Create" dropdown menu, we can select different types of rectangles. We will be using the center point one here.

- For sketches, we preferably want them to be fully constrained, usually to the origin. Constraining basically "locks" the sketch, so that the sketch cannot be accidentally altered and is indicated by black lines. Hence, the center point of the rectangle is right smack at the origin. This makes it constrained.

- To specify dimensions, you can key in values into the box which appears when you apply an item. For multiple values, you can use [Tab] to alternate between them.

- We shall now add the remaining shapes.

- Here, we added some construction lines for the inner rectangle, and trimmed the rectangle and the inside of the outermost circle. Trimming the rectangle however caused the sketch to be unconstrained.

- After adding the final rectangle, we are now done with the sketch! Unfortunately, it is not constrained hence the blue lines, but I am still able to extrude the sketch. This however has the risk of causing issues further down if sudden changes are made, or as Fusion calls it, "downstream failures".

- In any case, this exercise is done! On to the next one!

Exercise 2

- For this exercise, we are going to make this spanner looking model. This is where the fun begins.

- The very first step would be to make a sketch with reference to the one above.

- This may look complicated, but by breaking down the sketch down to basic shapes and using construction lines, it is very much doable.

- To break it down, the sketch first starts off with a circle with a pentagon within. Then, using a construction line, the second circle is made followed by the hexagon within. The circles are then connected with lines, and the inner shape is drawn.

- Now it is time to extrude. Click on Finish Sketch, then click on "Extrude".

- When extruding, a menu and a box pops up. You can edit certain settings in the menu, and you can input the value of how much you would like to extrude by using the box or the menu. Usually a face (the sketch) would already be highlighted and ready to extrude, but in some cases you will need to manually click where you would like to extrude. To select multiple faces, shift click. Once done, press [Enter] or the "OK" button in the menu to complete.

- Since the schematic did not specify the height / how much to extrude by, I roughly extruded by ~10-15mm.

- One final detail to note would be the fillets (curved edges) between the circles and the rectangle. You could draw them out in the sketch, but the more convenient method would be to use the Fillet command. You can specify the radius of the fillets, but in this case, since it isn't specified, we can eyeball an appropriate value.

- Complete!

Exercise 3

- The next thing we are going to make is a 2x4 Lego brick. This may look daunting, but thankfully there is a Youtube tutorial we can use as a reference.

- And here is the schematic for the brick.

- As always, we start off with a sketch.

- Then, extrude by 9.6 mm. We now have a cuboid as the base of the brick.

- Now, we are going to make the studs. One convenient method of making this feature would be to use the rectangular pattern command, which allows you to instantaneously draw out multiple of the same feature instead of creating each one manually.

- Again, we extrude each one to the required height of 1.8 mm.

- Now we need to create the insides of the brick. To do this, we are going to use the Shell feature on the bottom of the brick. This makes a solid model into an open shell, and you can specify the thickness of the shell wall as well. In this case we are going to specify it as 1.20 mm.

- Now for the final feature. We are going to start a sketch on the inner surface of the brick, and sketch out concentric circles for the bottom "pins". To accurately pinpoint the center, we can use a construction line from the top studs and snap the circles to the center of the line.

- We shall now extrude the bottom pin, then use the Rectangular pattern function once again to create the remaining two pins from the first one.

- By right, we are technically done. But just for an extra touch, we are going to use the fillet command on the edges of the studs and the brick to create small rounded edges so that the edges aren't so sharp.

- And we're done! This model can now be used for 3D printing and such.
